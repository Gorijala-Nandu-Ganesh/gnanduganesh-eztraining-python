List:-

 -->ordered
 -->duplicate items allowed
 -->items added at the end of the list

 1)append()- add an element to end of list
 2)extend()- add more than one elements at the end of the list
 3)insert()- insert an item at defined in dex
 4)remove()- removes an item from the list
 5)pop()- removes and returns an element at given index
 6)clear()- removes all items from list
 7)index()- returns index of first matched items
 8)count()- return count of number of items passed as an argument
 9)sort()- sort items in ascending order
 10)reverse()- reverse the order of the list
 11)copy()- returns a shallow copy of list
